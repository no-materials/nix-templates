# Nix templates

## Usage

### Templates

```console
nix flake init -t github:no-materials/nix-templates#bevy
```

or to specify a directory

```console
nix flake new -t github:no-materials/nix-templates#bevy ./test
```


### Dev shells

```console
nix develop github:no-materials/nix-templates#bevy
```

## List

```console
nix flake show
```
